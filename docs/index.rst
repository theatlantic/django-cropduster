Documentation
=============

django-cropduster is a project that makes a form field available that uses the `Jcrop jQuery plugin <https://github.com/tapmodo/Jcrop>`_. It is a drop-in replacement for Django's ``ImageField`` and allows users to generate multiple crops from images, using predefined sizes and aspect ratios. django-cropduster was created by developers at `The Atlantic <http://www.theatlantic.com/>`_.

Compatibility
=============

django-cropduster is compatible with python 2.7 and 3.4, and Django versions 1.4 - 1.8.
